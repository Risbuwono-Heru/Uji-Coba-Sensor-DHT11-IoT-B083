Repository ini merupakan kumpulan hasil praktikum dan tugas kelompok pada mata kuliah Internet of Things. Seluruh file disusun berdasarkan topik tugas agar memudahkan proses dokumentasi, penilaian, dan penelusuran isi project.

## Daftar Folder

### 1. `01-Demo-Display-OLED-SSD1306`
Folder ini berisi project demonstrasi penggunaan display OLED SSD1306 pada mikrokontroler. Di dalam folder ini terdapat source code program, dokumentasi hasil pengujian, serta file pendukung yang digunakan untuk menampilkan data atau teks pada layar OLED.

### 2. `02-Bacaan-Sensor-DHT-Melalui-WiFi`
Folder ini berisi project pembacaan sensor DHT11 menggunakan board ESP8266. Data suhu dan kelembapan ditampilkan pada OLED serta dapat diakses melalui web server. Folder ini memuat source code, dokumentasi hasil tampilan web, dan rangkaian hardware.

### 3. `03-Desain-Sistem-Sensor`
Folder ini berisi dokumen desain sistem sensor yang disusun sebagai pengganti ETS. Isi folder mencakup file PDF utama beserta diagram atau file pendukung lain yang menjelaskan rancangan sistem, alur kerja, dan komponen yang digunakan.

## Isi Repository
- Demo display OLED SSD1306
- Bacaan sensor DHT11 melalui WiFi
- Desain sistem sensor sebagai pengganti ETS

## Tujuan
- Mendokumentasikan hasil tugas dan praktikum kelompok
- Menyusun source code, dokumentasi, dan laporan secara rapi
- Memudahkan proses pengumpulan tugas melalui GitHub

## Catatan
Setiap folder memiliki file project, dokumentasi, dan file pendukung sesuai dengan topik tugas masing-masing. Untuk menjalankan source code berbasis ESP8266, pastikan library yang dibutuhkan sudah terpasang dan konfigurasi WiFi telah disesuaikan pada kode program.
